# 🚀 MacroBasic

Extensão de navegador desenvolvida para agilizar e padronizar atendimentos de help desk.

## 📌 Sobre o projeto

O **MacroBasic** permite criar e gerenciar mensagens prontas (macros), facilitando o envio rápido de respostas durante atendimentos via chat.

Com apenas um comando, é possível inserir automaticamente uma mensagem completa no campo de texto.

### 💡 Exemplo de uso
Digite no chat: /nome


➡️ A macro correspondente será inserida automaticamente.

---

## ⚙️ Funcionalidades

- ✅ Cadastro de macros personalizadas  
- ✅ Inserção rápida via comando `/nome`  
- ✅ Inserção via botão da extensão  
- ✅ Edição e exclusão de macros  
- ✅ Backup das macros (exportação em JSON)  
- ✅ Importação de macros  
- ✅ Atalho de teclado configurável  

---

## 🧩 Instalação (Modo Desenvolvedor)

1. Baixe ou clone este repositório  
2. Extraia os arquivos (deixe apenas **uma pasta principal**)  
3. Abra o navegador e acesse:  
   `chrome://extensions/`  
4. Ative o **Modo desenvolvedor**  
5. Clique em **"Carregar sem compactação"**  
6. Selecione a pasta do projeto  

---

## ⌨️ Configurar Atalho

1. Acesse `chrome://extensions/shortcuts`  
2. Localize a extensão **MacroBasic**  
3. Defina o atalho desejado  

---

## 🛠️ Como usar

### ➕ Criar macro
1. Abra a extensão  
2. Informe:
   - Título da macro  
   - Texto da mensagem  
3. Clique em **Salvar**

---

### ▶️ Utilizar macro

No campo de texto do chat:

- Digite `/nome-da-macro`  
ou  
- Use o botão da extensão  

➡️ A mensagem será inserida automaticamente

---

### ✏️ Gerenciar macros

- Editar macros existentes  
- Remover macros  
- Organizar conforme necessidade  

---

### 💾 Backup e restauração

- **Exportar:** baixa um arquivo `.json` com todas as macros  
- **Importar:** permite restaurar macros previamente salvas  

---

## 🎯 Objetivo

O objetivo do projeto é aumentar a produtividade no atendimento, reduzindo o tempo de digitação e garantindo padronização nas respostas.
